setAdvertisingIdCollectionEnabled()method
public function setAdvertisingIdCollectionEnabled(enabled:Boolean):void

Enables Advertising Id Collection. This will not work on iOS unless you also add the GAIDFAAccess.ane to your project! Also note that if you do that and use this function, you must also have AdMob or other advertising in your application or Apple will reject your app. On Android, they don't care what you do.

Parameters

enabled:Boolean — whether advertising id collection should be enabled

setTrackerField()method 
public function setTrackerField(fieldName:String, fieldValue:String):void

Sets a custom field to be used on all hits sent by this tracker. For instance, setTrackerField(GAFields.USER_ID,'bob123'); would associate all future hits on this tracker with the user bob123.

Parameters

fieldValue:String — the string value to set for this field
