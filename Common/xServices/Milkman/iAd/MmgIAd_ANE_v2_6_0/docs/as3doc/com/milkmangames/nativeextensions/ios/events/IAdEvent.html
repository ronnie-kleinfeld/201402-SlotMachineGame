Property Detail
willLeaveApplicationproperty
public var willLeaveApplication:Boolean

In a BANNER_ACTION_BEGIN event, this will be true if the user is going to be leaving your application to go to the app store or other external area. If false, the user will be interacting with a fullscreen takeover, and you should stop sound and pause important interactions until BANNER_ACTION_FINISHED is dispatched.
