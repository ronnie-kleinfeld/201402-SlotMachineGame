Fill the screen!

This solution works well for applications, like games, that do not accept text input. In our testing we have discovered some odd behavior that will have to be accounted for on various platforms.

  • On Android-based devices the keyboard can cover a text field when the text field is clicked
  • On iOS devices if a text field has focus and the keyboard is visible and the device is rotated the keyboard and scaling behave strangely

Another thing you'll notice is that on most devices this text won't show up. This is not a general-case solution, but rather a way of scaling up a simple game to fill the screen.
