Class Overview

The MapView class displays a map based on mapping data provided by the Amazon Maps API. You can define your MapView as an element in the XML file that specifies your app's layout. Alternately, you can define a MapView with one of the MapView constructors, then add the MapView to a layout View with ViewGroup.addView(View).

All MapView objects are displayed in a MapActivity.

This class is not thread-safe, except for getOverlays() and all modifications to the returned list of Overlays.

public boolean canCoverCenter ()

Returns whether there are tiles available to render the center of the map.

Returns
  • True if there are tiles in memory that correspond to the map's current center, false if not.

public void computeScroll ()

Calculates the intermediate steps of an animation, such as one initiated by a call to MapController.animateTo. Overriding computeScroll() is typically not needed and is for advanced use only. If you override, be sure to call super.computeScroll.

public void displayZoomControls (boolean takeFocus)

Shows the map's zoom control and, optionally, gives it focus for user input. If you have previously called setBuiltInZoomControls(true), displayZoomControls shows that control. Otherwise, displayZoomControls shows the zoom control that is returned from getZoomControls.

Parameters
takeFocus If true, automatically gives focus to the zoom control when it is displayed.

public ViewGroup.LayoutParams generateLayoutParams (AttributeSet attrs)

Creates a new LayoutParams object from an XML attribute set.

Parameters
attrs The XML attribute set from which to generate the LayoutParams.
Returns
  • The newly created LayoutParams object.

public MapController getController ()

Returns the map's MapController. You use the MapController to programmatically control panning and zooming of the map. The returned MapController is reuseable; you do not need to make repeat calls to getController for a given map.

Returns
  • The MapController associated with the current MapView.

public int getLatitudeSpan ()

Returns the difference in microdegrees between the latitude of the top edge of the MapView and the latitude of the bottom edge. Microdegrees are standard decimal geocoordinates multiplied by one million.

Returns
  • The range of latitude shown in the map, in microdegrees.

public int getLongitudeSpan ()

Returns the difference in microdegrees between the longitude of the left edge of the MapView and the longitude of the right edge. Microdegrees are standard decimal geocoordinates multiplied by one million.

Returns
  • The range of longitude shown in the map, in microdegrees.

public GeoPoint getMapCenter ()

Returns a point specifying the latitude and longitude of the center of the map.

Returns
  • A GeoPoint identifying the center of the map.

public int getMaxZoomLevel ()

Returns the maximum available zoom level for the map's current location. The maximum zoom level reflects the availability of data, based on the center point of the map.

Returns
  • The maximum available zoom level for the map.

public final List<Overlay> getOverlays ()

Returns the overlay list for the map. To add overlays to a map, you must first call getOverlays to obtain the list to which they are added. The overlay list is synchronized by Collections.synchronizedList(java.util.List), and the overlays are listed in the order in which they are drawn. Loops over the list should also be synchronized on the list. This method is thread-safe.

Returns
  • The map's synchronized overlay list.

public Projection getProjection ()

Returns a Projection instance that you can use to translate between real-world latitude and longitude (expressed in GeoPoints) and screen pixel coordinates.

Returns
  • A Projection for the map.

public View getZoomControls ()

Deprecated. Use setBuiltInZoomControls(boolean) instead.

Returns
  • A view containing the map's zoom control that can be added to the MapView or elsewhere in a layout. The zoom control is shown when the user touches the map, and after a period of inactivity are hidden again.

public int getZoomLevel ()

Returns the current zoom level of the map.

Returns
  • An integer identifying the zoom level of the map. At zoom level 1, the map is fully zoomed out to display the entire world. Greater zoom level values correspond with higher magnifications of the map content.

public boolean isSatellite ()

Returns whether the map is currently displaying satellite data.

Available for interface compatibility only. The traffic view setting is ignored and has no impact on map display.

public void onFocusChanged (boolean hasFocus, int direction, Rect unused)

Called when the MapView gains or loses focus. Stops any current panning on the map.

Parameters
hasFocus True if the MapView now has focus; otherwise, false.
direction The direction focus has moved. Possible values are FOCUS_UP, FOCUS_DOWN, FOCUS_LEFT, FOCUS_RIGHT, FOCUS_FORWARD, and FOCUS_BACKWARD. Provide this value when the focus changes due to directional navigation. If direction doesn't apply, ignore this value.

public boolean onKeyDown (int keyCode, KeyEvent event)

Handles key down events by passing them to the map's overlays. onKeyDown() presents events to visually higher overlays prior to visually lower overlays. That is, it passes events to overlays in reverse of the order in which they appear in the getOverlays() list. If an overlay handles the event, the underlying overlays do not receive the event. If none of the overlays handles the event, it is passed back up to MapView's superclass.

Parameters
keyCode An integer identifying the keyboard key that was pressed.
event The key down event to be handled.
Returns
  • A boolean value indicating whether the event has been handled.

public boolean onKeyUp (int keyCode, KeyEvent event)

Handles key up events by passing them to the map's overlays. onKeyUp() presents events to visually higher overlays prior to visually lower overlays. That is, it passes events to overlays in reverse of the order in which they appear in the getOverlays() list. If an overlay handles the event, the underlying overlays do not receive the event. If none of the overlays handles the event, it is passed back up to MapView's superclass.

Parameters
keyCode An integer identifying the keyboard key that was released.
event The key up event to be handled.
Returns
  • A boolean value indicating whether the event has been handled.

public void onRestoreInstanceState (Bundle state)

Retrieves the state of the current MapView from the specified Bundle. The state consists of the map's saved center point, zoom level, and the state of the built-in zoom control.

Parameters
state The Bundle from which to retrieve the MapView state.

public void onSaveInstanceState (Bundle state)

Saves the current state of the MapView to the specified Bundle object. The state consists of the map's current center point, zoom level, and the state of the built-in zoom control.

Parameters
state The Bundle to which the MapView state is added.

public boolean onTouchEvent (MotionEvent event)

Handles touch events by passing them to the map's overlays. If the MapView is clickable or enabled, onTouchEvent() presents events to visually higher overlays prior to visually lower overlays. That is, it passes events to overlays in reverse of the order in which they appear in the getOverlays() list. If an overlay handles the event, the underlying overlays do not receive the event. If none of the overlays handles the event, it is used to pan or zoom the MapView.

Parameters
event The touch event to be handled.
Returns
  • A boolean value indicating whether the event has been handled. False if the event is not handled or if the MapView currently is not enabled or not clickable.

public boolean onTrackballEvent (MotionEvent event)

Handles motion events by passing them to the map's overlays in their onTouchEvent(). onTrackballEvent() presents events to visually higher overlays prior to visually lower overlays. That is, it passes events to overlays in reverse of the order in which they appear in the getOverlays() list. If an overlay handles the event, the underlying overlays do not receive the event. If none of the overlays handles the event, nothing happens.

Parameters
event The motion event to be handled.
Returns
  • A boolean value indicating whether the event has been handled.

public void onWindowFocusChanged (boolean hasFocus)

Called when the window containing the MapView gains or loses focus. Stops any current panning.

Parameters
hasFocus True if the window now has focus; otherwise, false.

public void preLoad ()

Starts downloading map tiles prior to the MapView becoming visible. It is not typically necessary to call preLoad(), since tiles begin downloading as soon as the MapView is visible. This method is only needed if you want to start the downloading process earlier.

public void setBuiltInZoomControls (boolean on)

Enables or disables the built-in zoom control. When enabled, the zoom control is shown when the user touches the screen, then hidden after a period of inactivity. Use setBuiltInZoomControls instead of getZoomControls(). The zoom control returned by getZoomControls is not hidden if the built-in zoom control is enabled.

Parameters
on If true, the built-in zoom control is shown automatically.

protected final void onDraw (Canvas canvas)

Draws onscreen all necessary map tiles, including any animations. Also draws any overlays that are currently in the getOverlays() collection for the map.

This method calls draw(Canvas, MapView, boolean, long) twice for each overlay in the getOverlays() list. The first call is for drawing the shadows and other features behind the primary content of the overlay, and the second call is for drawing the primary content.

canvas The Canvas object on which to draw.
