Class Overview

Inheriting from ViewGroup.LayoutParams the MapView.LayoutParams contain additional layout information specific to aligning child views on a map. Depending on the mode, a child view can be positioned relative to the view (MODE_VIEW) or relative to the underlying map (MODE_MAP).

Fields

public int alignment

The alignment field specifies which point on the child view will be aligned to either the underlying map or to the view. The low order byte is the horizontal alignment and the higher is vertical. A vertical and horizontal alignment constant can be or'ed together But do not or together more than one of each. CENTER, BOTTOM_CENTER and TOP_LEFT represent alignments in both directions.

public int mode

The layout mode. Set to either MODE_VIEW or MODE_MAP

public GeoPoint point

The location on the underlying map that the child view will align to. This is used when mode is set to MODE_MAP.

public int x

The x offset for alignment. In MODE_VIEW this represents the horizontal distance in pixels between the top left corner of the MapView and the alignment point on the child view. In MODE_MAP, this is the horizontal distance in pixels from the specified point on the map and the alignment point on the child view.

public int y

The y offset for alignment. In MODE_VIEW this represents the vertical distance in pixels between the top left corner of the MapView and the alignment point on the child view. In MODE_MAP, this is the vertical distance in pixels from the specified point on the map and the alignment point on the child view.

public MapView.LayoutParams (int width, int height, GeoPoint point, int alignment)

This creates a new set of Mapvew.LayoutParams. This specifies alignment in MODE_MAP and will place the alignment point exactly onto the specified GeoPoint point.

Parameters
width The width of the child view. Set to FILL_PARENT, WRAP_CONTENT or an exact size in pixels.
height The height of the child view. Set to FILL_PARENT, WRAP_CONTENT or an exact size in pixels.
point The point on the map to place the alignment point on.
alignment The point on the child view to place on the point. See alignment

public MapView.LayoutParams (int width, int height, GeoPoint point, int x, int y, int alignment)

This creates a new set of Mapvew.LayoutParams. This specifies alignment in MODE_MAP and will place the alignment point x pixels below and y pixels to the right of the specified GeoPoint point.

Parameters
width The width of the child view. Set to FILL_PARENT, WRAP_CONTENT or an exact size in pixels.
height The height of the child view. Set to FILL_PARENT, WRAP_CONTENT or an exact size in pixels.
point The point on the map that the layout will start from.
x The horizontal offset in pixels from point to alignment.
y The vertical offset in pixels from point to alignment.
alignment The position on the child view that will be placed (x,y) away from point. See alignment

public MapView.LayoutParams (int width, int height, int x, int y, int alignment)

This creates MapView.LayoutParams similar to an AbsoluteLayout. The only difference is that it is the alignment point of the child view that is placed at offset (x,y)in the parent rather than simply the TOP_LEFT. This sets mode to MODE_VIEW.

Parameters
width The width of the child view. Set to FILL_PARENT, WRAP_CONTENT or an exact size in pixels.
height The height of the child view. Set to FILL_PARENT, WRAP_CONTENT or an exact size in pixels.
x The horizontal offset in pixels from the top left of the MapView to alignment.
y The vertical offset in pixels from the top left of the MapView to alignment.
alignment The position on the child view that will be placed at (x,y) in the MapView. See alignment

public MapView.LayoutParams (ViewGroup.LayoutParams source)

Copy Constructor Copies ViewGroup parameters for height and width, and sets the mode to MODE_VIEW and alignment to TOP_LEFT. This will place the child view in the top left corner of the MapView.
