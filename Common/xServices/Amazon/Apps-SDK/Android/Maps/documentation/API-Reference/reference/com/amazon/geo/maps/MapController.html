public void animateTo (GeoPoint point, Message message)

Starts a one second animation towards the given GeoPoint. Once the animation has completed the given GeoPoint will be the center of the map and the given Message is dispatched to it's target. If the animation is aborted, the given Message is null or no target is specified in the given Message, no action is taken.

Parameters
point The GeoPoint to animate towards
message The Message to dispatch once the animation has successfully completed

public void animateTo (GeoPoint point, Runnable runnable)

Starts a one second animation towards the given GeoPoint. Once the animation has completed the given GeoPoint will be the center of the map and the given Runnable will be executed on the UI thread. If the animation is aborted or the given Runnable is null, no action is taken.

Parameters
point The GeoPoint to animate towards
runnable The Runnable to execute in the UI thread once the animation has completed

public int setZoom (int zoomLevel)

Set the zoom level of the map to the given integer. This value will be clipped 1 and 21 (inclusive). There is no animation associated with this action.

Note that not all areas have tiles at higher zoom levels. In these areas, a place-holder tile is shown.

Parameters
zoomLevel The zoomlevel you wish to set.
Returns
  • 22 if you attempted to set the zoom to a value above 21, otherwise the new zoom level.

public void stopAnimation (boolean jumpToFinish)

Stops any animation that may be in progress, and conditionally moves the map center to the initially requested endpoint.

Parameters
jumpToFinish - If true, the map center will jump to the initially requested end point. If false, the map center that the animation has currently achieved will be the final map center.

public boolean zoomIn ()

This starts a one second animated zoom-in based on the center of the map. If you have added ZoomControls from getZoomControls(), then a call to displayZoomControls(boolean) will be made causing the ZoomControls to appear briefly.

Returns
  • true if we started the animated zoom-in, false if we cannot zoom in further.

public boolean zoomInFixing (int xPixel, int yPixel)

Parameters

Returns
  • true if we started the animated zoom-in, false if we cannot zoom-in any further.
