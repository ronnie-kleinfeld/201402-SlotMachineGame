Class Overview

A single item on an map overlay, and the basic component of an ItemizedOverlay. Overlay items include:

  • A location (GeoPoint).
  • An optional marker Drawable.
  • A short title.
  • A longer description (snippet).

In addition, an overlay item can have a state (or a combination of states): it can be pressed, focused or selected. Different marker drawables can be returned based on current state of that item.

The features of an overlay item are managed by its enclosing ItemizedOverlay object. Use the ItemizedOverlay.onTap() method to change the item's marker or display its title or description in a pop-up.

public Drawable getMarker (int stateBitset)

Returns the marker Drawable to indicate this item on the map overlay. If the marker is null, the default marker from ItemizedOverlay is drawn instead.

Overlay items can have multiple states (pressed, focused, selected), and the marker drawable for an item can include images for the different states. The state of the overlay item is defined by a bitset that uses the flags ITEM_STATE_PRESSED_MASK, ITEM_STATE_SELECTED_MASK, or ITEM_STATE_FOCUSED_MASK.

The default implementation of this method calls {link #setState(Drawable, int)} on the item's marker drawable(if not null) with the bitset, and returns the marker drawable.

Parameters
stateBitset the bitset for the item's state.
Returns
  • the drawable object for the given marker state.
