Class Overview

A map Overlay that indicates the user's current location and an optional compass indicator. MyLocationOverlay uses the orientation sensor and LocationManager to determine the user's location and bearing (direction).

In your activity's onResume() method, call enableMyLocation() and/or enableCompass() to enable the location or compass overlay.

In your activity's onPause() method, call disableMyLocation() and/or disableCompass() to stop listening for location and orientation data when your app is running in the background.

You can provide a Runnable in the runOnFirstFix() method, which is invoked when the LocationManager reports a location fix.

Override dispatchTap() in your subclass to manage taps on the location indicator.

This class is thread-safe, unless otherwise noted.

public boolean draw (Canvas canvas, MapView mapView, boolean shadow, long when)

Draw the location overlay with the user's current location and compass indicator (if enabled and available). No items are drawn on the shadow pass. This method must run on the main thread.

If the orientation sensor is enabled (that is, enableCompass() has been invoked), the drawCompass() method is invoked to draw the compass indicator.

If location is enabled (enableMyLocation()) and the user's location is available, the drawMyLocation() method is invoked to draw a location and accuracy pointer.

Parameters
canvas The Canvas to draw upon.
mapView The MapView that requested the draw.
shadow A boolean value that is true for the shadow draw pass, and false for the pass that draws the overlay content.
when The timestamp of the draw call in milliseconds, as returned by currentAnimationTimeMillis() at at the time of the call.
Returns
  • false
See Also
  • drawCompass(Canvas, float)
  • drawMyLocation(Canvas, MapView, Location, GeoPoint, long)

public boolean runOnFirstFix (Runnable runnable)

Registers a Runnable object that is executed when the location manager (LocationManager) provides a location fix.

If a location fix is already available, the runnable is executed immediately and returns true. Otherwise this method returns false and the runnable is executed in a new thread once the location fix is available.

Parameters
runnable the runnable to execute when the location is available.
Returns
  • true if the runnable ran now, false otherwise.

protected void drawMyLocation (Canvas canvas, MapView mapView, Location lastFix, GeoPoint myLocation, long when)

Draws a location indicator at, and an accuracy indicator around, the user's most recent location point. The accuracy indicator is a circle around the location point, where the radius of that circle indicates the accuracy of the location fix.

Invoked from the draw() method. This method must run on the main thread.

Override this method to change the appearance of the location indicator.

Parameters
canvas the Canvas to draw on.
mapView the MapView that invoked the draw.
lastFix the last location received from the LocationManager.
myLocation same as lastFix, but in GeoPoint form.
when The drawing time, in milliseconds, as derived from currentAnimationTimeMillis().
