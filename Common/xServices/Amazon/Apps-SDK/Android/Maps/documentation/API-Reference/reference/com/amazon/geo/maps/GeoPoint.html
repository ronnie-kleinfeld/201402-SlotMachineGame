Class Overview

A point on the Earth depicted by a latitude and longitude pair. Both latitude and longitude are expressed in microdegrees. Microdegrees are decimal degrees multiplied by one million. For example, if the decimal degree value were 34.567890, that value in microdegrees would be 34567890.

Note that this class is immutable.

Public Constructors

public GeoPoint (int latitudeE6, int longitudeE6)

Constructs a GeoPoint with the given latitude and longitude. Both the latitude and longitude are expressed in microdegrees.

You can convert decimal degrees (float) to microdegrees (int) like this:

GeoPoint gp = new GeoPoint((int)(lat * 1E6), (int)(lng * 1E6));

Parameters
latitudeE6 The latitude, in microdegrees. The latitude is clipped to values between -80 and 80 (inclusive), in order to maintain accuracy in the Mercator projection.
longitudeE6 The longitude, in microdegrees. The longitude is normalized to values between -180 (exclusive) and 180 (inclusive). If longitudeE6 is -540 degrees, -180 is used instead.
