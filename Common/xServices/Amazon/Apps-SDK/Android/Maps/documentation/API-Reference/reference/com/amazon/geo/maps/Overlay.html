Class Overview

Base class for overlaying content on a map. Extend this class if you want to create an overlay to show transparency effects, art, or other visual content on top of a map. To show discrete items such as pins for points of interest, use the ItemizedOverlay subclass instead.

To add an overlay to your MapView, add the overlay to the overlay list obtained by getOverlays().

public boolean draw (Canvas canvas, MapView mapView, boolean shadow, long when)

Draws the overlay on the MapView and supports animation with the when parameter. When the draw() method is called, the when parameter receives a timestamp equal to the current animation time in milliseconds as returned by currentAnimationTimeMillis().

This method is always called twice during each call to onDraw(), as follows. Note that the when parameter receives the same timestamp for each pass.

  1. draw() is called with the shadow parameter set to true. This step is for drawing the shadows and other features that will be behind the primary content of the overlay. If your overlay has no shadows and no features behind the primary content, you can return immediately when shadow is true.
  2. draw() is called with the shadow parameter set to false. This step is for drawing the primary content of the overlay.

The default implementation calls through to draw(Canvas, MapView, boolean).

Parameters
canvas The Canvas to draw upon.
mapView The MapView that requested the draw.
shadow A boolean value that is true for the shadow draw pass, and false for the pass that draws the overlay content.
when The timestamp of the draw call in milliseconds, as returned by currentAnimationTimeMillis() at at the time of the call.
Returns
  • true if an immediate redraw is needed; false otherwise.

protected static void drawAt (Canvas canvas, Drawable drawable, int x, int y, boolean shadow)

Draws the given Drawable on the Canvas at the given offset in pixels. You can obtain the offset by calling toPixels(GeoPoint, Point).

Parameters
canvas The Canvas on which to draw.
drawable The Drawable to draw.
x The x coordinate, in pixels, of the offset.
y The y coordinate, in pixels, of the offset.
shadow A boolean value that is true for the shadow draw pass, and false for the overlay draw pass. When shadow is true, you should draw the shadows or features behind the Drawable, or return immediately if there are no shadows or features behind the Drawable.
