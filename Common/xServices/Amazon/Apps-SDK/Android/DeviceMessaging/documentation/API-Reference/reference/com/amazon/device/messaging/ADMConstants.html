Constants

public static final String ERROR_AUTHENTICATION_FAILED

Passed to onRegistrationError(String) when an authentication failure occurs. The user may need to re-register their device with Amazon.

Constant Value: "AUTHENTICATION_FAILED"

public static final String ERROR_INVALID_SENDER

Passed to onRegistrationError(String) when the ADM service did not recognize your API Key. Confirm that your API Key is embedded in your APK, according to these instructions and that you have successfully obtained valid ADM credentials.

Constant Value: "INVALID_SENDER"

public static final String ERROR_SERVICE_NOT_AVAILABLE

Passed to onRegistrationError(String) when the ADM service is unavailable. If you are using ADMMessageHandlerBase to receive your messages, you should not retry, as it has already done so.

Constant Value: "SERVICE_NOT_AVAILABLE"

public static final String EXTRA_MD5

An extra in the Intent provided to onMessage(android.content.Intent) containing the MD5 hash of the message payload.

Constant Value: "adm_message_md5"
