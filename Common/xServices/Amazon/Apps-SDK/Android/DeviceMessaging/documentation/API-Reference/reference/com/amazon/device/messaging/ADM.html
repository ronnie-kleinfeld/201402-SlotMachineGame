Class Overview

This class provides the core methods to support registering an app instance to use Amazon Device Messaging (ADM).

Note: Although ADM methods are thread-safe, they are not process safe. You should avoid using ADM from different processes of the same app simultaneously.

Public Methods

public String getRegistrationId ()

Obtains the current registration ID for this instance of your app.

Returns
  • The current registration ID for this app instance or null if the current instance is not registered.
Throws
IllegalStateException if ADM is not supported on the device.

public boolean isSupported ()

Indicates whether ADM is supported on the current device.

Returns
  • true if ADM is present.

public void startRegister ()

Starts an asynchronous operation that registers your app for receiving ADM messages. Your ADMMessageHandlerBase receives a onRegistered(String) callback when startRegister() completes or an onRegistrationError(String) callback if it fails.

Throws
IllegalStateException if ADM is not supported on the device.

public void startUnregister ()

Starts an asynchronous operation that unregisters your app from receiving ADM messages. Your ADMMessageHandlerBase receives an onUnregistered(String) callback when startUnregister completes or an onRegistrationError(String) callback if it fails.

Throws
IllegalStateException if ADM is not supported on the device.
