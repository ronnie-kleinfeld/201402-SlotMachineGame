PURCHASES_UPDATEDConstant 
public static const PURCHASES_UPDATED:String = azPurchaseUpdated

Dispatched when the status of purchased Items has been updated for the user. This event may be dispatched any time after the SDK is initialized. If any SKUs have been revoked since the last time the extension was initialized, they will be contained in the revokedSkus property. If there are any receipts available for currently updated purchases, they will be contained in the receipts property.

SDK_AVAILABLEConstant 
public static const SDK_AVAILABLE:String = azSdkAvailable

Dispatched After the Amazon Purchase SDK has been initialized by calling AmazonPurchase.create(). The event property 'isSandboxMode' will be TRUE if the app is currently in sandbox/testing mode, or FALSE if the app is currently in production mode (live on the Amazon App Store.)
