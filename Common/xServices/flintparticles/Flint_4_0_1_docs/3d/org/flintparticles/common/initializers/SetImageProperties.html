Packageorg.flintparticles.common.initializers
Classpublic class SetImageProperties
InheritanceSetImageProperties InitializerBase Object

The SetImageProperties Initializer can be used to set any properties of a particle's image object. This Initializer has a single property that is any object whose properties should be copied to the image object. This Initializer has a priority of -10 so that it runs after the image setting Initializers, which have a priority of zero.
