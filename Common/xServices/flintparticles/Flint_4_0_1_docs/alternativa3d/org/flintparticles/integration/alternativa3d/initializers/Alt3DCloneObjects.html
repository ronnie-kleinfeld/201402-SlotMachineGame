Classpublic class Alt3DCloneObjects
InheritanceAlt3DCloneObjects ImageInitializerBase InitializerBase Object

The A3D4CloneObjects Initializer sets the 3D Object to use to draw the particle in a 3D scene. It selects one of multiple objects that are passed to it and calls the clone method of the object to create the particle object. It is used with the Away3D 4 renderer when particles should be represented by a 3D object.

This class includes an object pool for reusing objects when particles die.

Alt3DCloneObjects()Constructor
public function Alt3DCloneObjects(objects:Array = null, weights:Array = null, usePool:Boolean = false, fillPool:uint = 0)

The constructor creates an A3D4CloneObjects initializer for use by an emitter. To add an A3D4CloneObjects to all particles created by an emitter, use the emitter's addInitializer method.

Parameters
objects:Array (default = null) — An array containing the Object3Ds to use for each particle created by the emitter.
 
weights:Array (default = null) — The weighting to apply to each Object3D. If no weighting values are passed, the images are used with equal probability.
 
usePool:Boolean (default = false) — Indicates whether particles should be reused when a particle dies.
 
fillPool:uint (default = 0) — Indicates how many particles to create immediately in the pool, to avoid creating them when the particle effect is running.
