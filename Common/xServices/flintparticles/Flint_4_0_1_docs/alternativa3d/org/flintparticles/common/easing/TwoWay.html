TwoWayMethods
 
Packageorg.flintparticles.common.easing
Classpublic class TwoWay
InheritanceTwoWay Object

A set of easing equations that start and end at the end value and reach the start value at the half-time point. They are designed for modifying the particle energy such that it starts and ends at zero and peaks at half way through the particle's lifetime.
