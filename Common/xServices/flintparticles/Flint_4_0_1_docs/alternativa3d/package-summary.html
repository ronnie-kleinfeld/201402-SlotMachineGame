A particle system in Flint is created by combining emitters, counters, initializers, actions, activities and renderers.

  • Emitters create and manage Particles.
  • Emitters use Counters to control how many particles are created.
  • Emitters use Initializers to control how those particles are initialized.
  • Emitters use Actions to control how those particles behave over time.
  • Emitters use Activities to control their own behaviour over time.
  • Renderers draw the Particles managed by one or more emitters.

Flint's Alternativa3D integration works with Alternativa3D version 8.5. It uses renderers that draw the particle effect within an Away3D scene.

If using the swc files, you will need

  • Flint3d.swc
  • FlintAlternativa3d.swc
