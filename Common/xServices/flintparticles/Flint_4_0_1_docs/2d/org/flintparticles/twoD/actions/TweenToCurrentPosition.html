Classpublic class TweenToCurrentPosition
InheritanceTweenToCurrentPosition ActionBase Object
Implements Initializer

The TweenToCurrentPosition action adjusts the particle's position between two locations as it ages. The start location is a random point within the specified zone, and the end location is the particle's position when it is created or added to the emitter. The current position is relative to the particle's energy, which changes as the particle ages in accordance with the energy easing function used. This action should be used in conjunction with the Age action.
